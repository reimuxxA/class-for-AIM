## 变量与数据类型

* 在Python中，变量是存储数据的容器。Python是动态类型语言，意味着你不需要显式声明变量类型。以下是基本数据类型的示例：
```python
#整数
number = 10

#浮点数
decimal = 3.14

#字符串
text = "Hello, Python!"

#布尔值
is_valid = True

```

## 运算符与表达式
* 运算符用于执行操作，表达式是由变量和运算符组成的语句。Python支持多种运算符：
    * 算术运算符：`+`,` -`,` *`,` /`,` //`,` %`,` **`
    * 比较运算符：`==`, `!=`, `>`, `<`, `>=`, `<=`
    * 逻辑运算符：`and`, `or`, `not`
    
## 算术运算符

算术运算符用于执行基本的数学运算。

| 运算符 | 描述                       | 示例               |
|--------|----------------------------|--------------------|
| +      | 加法                       | `a + b`            |
| -      | 减法                       | `a - b`            |
| *      | 乘法                       | `a * b`            |
| /      | 除法                       | `a / b`            |
| //     | 整除（地板除法）           | `a // b`           |
| %      | 取模（求余数）             | `a % b`            |
| **     | 幂运算（求指数）           | `a ** b`           |

## 比较运算符

比较运算符用于比较两个值，结果总是布尔值。

| 运算符 | 描述                       | 示例               |
|--------|----------------------------|--------------------|
| ==     | 等于                       | `a == b`           |
| !=     | 不等于                     | `a != b`           |
| >      | 大于                       | `a > b`            |
| <      | 小于                       | `a < b`            |
| >=     | 大于或等于                 | `a >= b`           |
| <=     | 小于或等于                 | `a <= b`           |

### 逻辑运算符

逻辑运算符用于组合布尔表达式。

| 运算符 | 描述                       | 示例                                   |
|--------|----------------------------|----------------------------------------|
| and    | 逻辑与                     | `a and b`<br>当a和b都为True时，结果为True。 |
| or     | 逻辑或                     | `a or b`<br>当a或b至少有一个为True时，结果为True。 |
| not    | 逻辑非                     | `not a`<br>如果a为True，结果为False；如果a为False，结果为True。 |

### 存在的问题

- 算术运算符中的`//`执行整除操作，返回不大于结果的最大整数。
- 幂运算符`**`可以用于计算幂次方，例如`2 ** 3`表示2的3次方。
- 逻辑运算符中的`and`和`or`遵循短路逻辑，即如果第一个操作数已经能够确定结果，第二个操作数将不会被评估

```python
a = 10
b = 20

a>b and XXX #不再判断，已经是False

a<b or XXX #不在判断，已经是True

```

```python
#算术运算
result = 10 + 5 * 2

#比较运算
is_equal = (10 == 10)

#逻辑运算
is_true = True and False
```

[^布尔类型]:布尔值是编程中的一种基本数据类型，用于表示真值（True）和假值（False）。在Python中，布尔值是`bool`类型的实例，可以直接使用`True`和`False`这两个关键字来表示。
