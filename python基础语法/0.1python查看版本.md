# 0.1python查看版本

我们可以在命令窗口(Windows 使用 win+R 调出 cmd 运行框)使用以下命令查看我们使用的 Python 版本：
如果你是linux or mac用户打开终端就可以了

```cmd or bash
python -V
或
python --version
```

输出结果
`Python 3.13.0`

* 也能进入Python的交互式编程模式，查看版本：
    * 方法一：搜索框搜索idle，进入（windows）
    * 方法二：在终端中输入python

